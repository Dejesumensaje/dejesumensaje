About the project

Alucine is a website to analyze what’s at the heart of the audiovisual productions that we love. It also has a podcast called Muñeca Rusa in which we’re going deep to create bridges between fictional stories and what’s happening around the world.

The challenge

Alucine is a brand new website, so we need to strenghten our brand through a consistent UI. In order to achieve this, I created a first proposal of a design system that I’m currently iterating.

UI Kit 1.0

Through benchmark, I’ve gathered references to build a UI clean and easy to read.

I like the mix between Serif and Sans-serif fonts. So my first step was to define a typography composition.




I’ve also made a color palette to help with brand consistency.




To create a good visual experience, I used a layout of 12 columns, easily replicable with bootstrap or bulma.
